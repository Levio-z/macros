# my_vec! 宏详解

## 1. 基本用法

```rust
let v = my_vec![1, 2, 3]; // Vec<i32>
let v = my_vec![];        // 空 Vec
let v = my_vec![1];       // 单元素
```

## 2. 宏规则语法

### 2.1 整体模式: `$($x:expr),*`

```
$()           - 捕获组定界符
$x:expr       - $变量名:类型 捕获一个表达式
,*            - 匹配零个或多个逗号分隔的重复
```

**匹配过程**:
- `1, 2, 3` → 捕获 `1`, `2`, `3` 三个表达式
- 可选尾随逗号 `1, 2, 3,` 也能匹配

### 2.2 重复模式: `$(v.push($x);)*`

```
$(...)*       - 重复捕获次数
$x            - 引用之前捕获的值
```

展开时，`*` 会重复展开，次数等于捕获组数量。

## 3. 完整展开过程

输入: `my_vec![1, 2, 3]`

| 步骤 | 说明 |
|------|------|
| 1 | `$($x:expr),*` 捕获 `1`, `2`, `3` |
| 2 | 生成 `let mut v = Vec::new();` |
| 3 | `$(v.push($x);)*` 展开 3 次 |
| 4 | 最终返回 `v` |

### 展开结果

```rust
{
    let mut v = Vec::new();
    v.push(1);
    v.push(2);
    v.push(3);
    v
}
```

## 4. Rust 内置 vec! 源码参考

标准库 `vec!` 宏（简化版）:

```rust
macro_rules! vec {
    ($($x:expr),*) => {
        {
            let mut v = Vec::new();
            $(v.push($x);)*
            v
        }
    };
    ($x:expr; $n:expr) => {
        vec![$x; $n]  // 另一种语法: vec![0; 5] -> [0,0,0,0,0]
    };
}
```

## 5. 语法要点总结

| 语法 | 含义 |
|------|------|
| `$var:name` | 捕获变量 |
| `expr` | 表达式类型 |
| `*` | 零个或多个 |
| `+` | 一个或多个 |
| `?` | 零个或一个 |
| `$(...)*` | 重复展开 |
